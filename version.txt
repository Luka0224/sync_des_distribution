[Sync_des v1.4.2 업데이트 패치노트] 배포일: 2026.01.08

================================

■ 주요 기능 추가 : 자막 전환 효과(Transition Effect)

자막 등장 애니메이션 3종 추가 기존에 텍스트가 즉시 바뀌던 방식 외에, 부드럽게 전환되는 효과들이 추가되었습니다.

None : 효과 없음 (기존과 동일)
Fade : 투명도 조절을 통해 가사가 서서히 나타나는 효과
Slide Up : 가사가 아래에서 위로 부드럽게 올라오는 효과
Zoom In : 가사가 약간 작았다가 커지면서 강조되는 효과

UI 변경 사항

프로그램 상단 메뉴(저장/업데이트 버튼 옆)에 '효과 선택' 메뉴가 추가되었습니다.
가사 재생 중에도 실시간으로 효과를 변경할 수 있습니다.

시스템 개선

가사 렌더링 엔진 수정 : 애니메이션 처리를 위한 내부 로직 최적화
화면 갱신 구조 개선
